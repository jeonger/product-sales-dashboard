# Product Sales Dashboard (Power BI + PostgreSQL)

This project is a hypothetical **Product Sales Anayltics Dashboard** built in **Power BI**, using data stored in **PostgreSQL** that is queried with **SQL**. This simulates a small retail and e-commerce company's sales data and demonstrates end-to-end BI skills:

  - Data modeling
  - SQL querying
  - DAX measures
  - Interactive dashboard design
  - Business-focused analysis

## Tech Stack

- **Database:** PostgreSQL
- **Query Tool:** pgAdmin
- **Visualization:** Power BI Desktop
- **Language:** DAX
- **Data Format:** CSV → imported into PostgreSQL

## Project Structure

product-sales-dashboard/
├── powerbi/
│   └── product_sales_dashboard.pbix     # Main Power BI report
├── data/
│   └── sales_data.csv                   # Source sales dataset
├── sql/
│   ├── create_table.sql                 # Table definition for sales_data
│   └── notes_and_queries.sql            # Example analysis queries
